# I Design 4 U - Premium Web Design Agency Website

## Project Overview
A premium futuristic website for "I Design 4 U" digital web design agency featuring dark theme with purple/black color scheme, glassmorphism UI, and modern animations.

## Original Problem Statement
Create a premium futuristic website for a digital web design agency with dark theme, purple gradients (#7B2FF7, #9F5BFF), glassmorphism cards, countdown timer, pricing toggle, portfolio with device mockups, SEO blog, contact form, and WhatsApp button.

## User Personas
- **Small Business Owners**: Looking for affordable professional websites
- **Startups**: Need modern web presence with limited budget
- **Local Businesses**: Gyms, salons, restaurants seeking online visibility
- **Entrepreneurs**: Personal brands and portfolios

## Core Requirements
- Dark theme with purple/black color palette
- Glassmorphism UI cards
- Framer Motion animations
- Mobile responsive design
- SEO-optimized blog
- Contact form with email notifications
